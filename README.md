# orange project

귤의 당도를 예측하는 AI 모델 제작

## 📌 프로젝트 개요
- 목적: 동아대학교 만다린 해커톤
- 기간: 2025.11.08 ~ 2025.11.13

## 🛠 사용 기술
Python, Pandas, Scikit-learn, PyTorch 등

## 📊 주요 결과
- 성공적인 결과물을 만들지는 못했음.

## 실패 원인
- 제대로된 데이터를 수집하는게 어려웠음.
- 하이퍼 파라미터 튜닝 스킬 부족.

## 🗂 폴더 구조
```
my-project/
├── README.md        
├── notebooks/       ← 디지털 상의 데이터를 이용해 모델을 만들었습니다.
├── src/             ← 직접 제작한 데이터를 이용해 모델을 만들었습니다.
├── data/            ← 샘플 데이터
└── requirements.txt ← 사용한 라이브러리 목록
```

## 🚀 실행 방법
```bash
pip install -r requirements.txt
python main.py
```
